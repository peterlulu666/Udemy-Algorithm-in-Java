Stack

The main method is the starting point in Java. We would declare variable age.

int age = 12;

We could define do work above main.

int age = 12;
do work();

There are variables inside the do work method.

do work(){
    int weight = 116;

}

There is do more work method inside do work.

do work(){
    int weight = 116;
    do moreWork(){

    }
}

Once do work done here, it will be eliminated from the stack.


Heap

We would declare variable name.

We would declare variable myCar. This variable is going to point to actual abject.

We would use new keyword to create an object and assign it to myCar. myCar is a reference variable. It dose not contain the object. It refers to the address of memory.

int age = 12;
do work();
String name = "";
// When we here, we do not have object
Car myCar;
// when we here, we have object
myCar = new Car;
// the old car will be eliminated from heap. 
// this is how Java garbage collection works.
myCar = new Car;
